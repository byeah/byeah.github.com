My research experience covered the areas of data visualization and image processing. Aim at providing high-quality visual content, my past work gave me sufficient experience on interface design, graphics computing and human perception analysis. I am also interesting in information retrieval techniques and would like to designing high efficient data processing system which can handle the big data challenge.

 

Interactive Visual Analysis of Large Scale Geographic Data using WebGL

Cooperated with Jeffrey Heer and Zhicheng Liu at Visualization Group, Stanford

Based on project Profiler, we apply WebGL to enable parallel computing. This new web technique greatly improves system efficiency for both data aggregation and rendering. When visualizing geographic data, we provide real time rendering performance comparing to static approach like Microsoft HotMap.

My summary poster here.

Accepted Paper: imMens: Enabling Real-time Visual Querying of Big Data (EuroVis 2013)


Inverse Image Editing

Cooperated with Kun Xu, Jue Wang, Shiming Hu at Graphics Computing Group

Given a source image and a edited image, we try to recover the editing process and apply such transform on other images.

Submitted paper: Recovering a Semantic Editing History from a Before-and-After Image Pair (SIGGRAPH AISA 2013)

 

Change Blindness Image

Cooperated with Liqian Ma, Kun Xu, Shiming Hu at Graphics Computing Group

Analyzing the interesting game "Find the differences of two image" and the human vision phenomenon change blindness, we did experiments to find out what kind of difference is hard to discover. We also purposed a novel saliency model to measure such difficulties and implemented a system to automatically generate pairs of different images.

Accepted Paper: Change Blindness Images (TVCG 2013 )
